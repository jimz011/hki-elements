# HKI Elements Card Suite

**A collection of custom Home Assistant cards and integrations for creating beautiful, functional dashboards.**

---

## What is HKI Elements?

HKI Elements is an ecosystem of interconnected custom cards and integrations for Home Assistant, designed to work seamlessly together. These components share a consistent design language, modern nested configuration structure, and powerful customization options.

All cards support Jinja2 templates, real-time updates, and can inherit styling from each other for a cohesive dashboard experience.
