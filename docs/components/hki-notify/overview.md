# HKI Notify Integration
A Home Assistant custom integration that creates notification sensors and provides services to send, update, and dismiss notifications displayed by the HKI Notification Card.

## Overview
The HKI Notify integration creates **sensor entities** that store notification messages. These sensors work with the **HKI Notification Card** to display notifications in your Home Assistant dashboard.

## Features

- **Multiple Sensors** - Create separate notification sensors for different areas (Kitchen, Living Room, etc.)
- **Persistent Notifications** - Messages persist across Home Assistant restarts
- **Flexible Actions** - Notifications can trigger navigation, services, or URLs
- **Rich Styling** - Per-notification colors, fonts, borders, and shadows
- **Confirmation Dialogs** - Optional confirmation before executing actions
- **Timestamp Tracking** - Automatic or manual timestamp assignment
