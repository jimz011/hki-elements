# Installation

### HACS (Recommended)

1. Open HACS in Home Assistant
2. Click on "Integrations"
3. Click the three dots menu â†’ "Custom repositories"
4. Add repository URL: `https://github.com/jimz011/hki-notify`
5. Select category: "Integration"
6. Click "Add"
7. Find "HKI Notify" and install it
8. Restart Home Assistant

### Manual Installation

1. Download the `hki_notify` folder from the repository
2. Copy it to your `custom_components` directory
3. Restart Home Assistant
