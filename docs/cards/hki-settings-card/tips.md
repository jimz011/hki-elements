# Tips & Tricks

- Place this card on a hidden/admin dashboard view to avoid accidental edits.
- Start with only a few global values (colors, radius, typography), then expand.
- Use per-card config only when a card should intentionally differ from the global style.
- If values seem stale, reopen the editor and save again to republish global settings.
- Use scoped reset before full reset to avoid losing unrelated defaults.
