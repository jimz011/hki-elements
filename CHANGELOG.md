## [1.0.0] - 2025-02-14

### Added
- Initial release of HKI Elements bundle
- HKI Header Card v2.0.5
- HKI Button Card v1.0.6
- HKI Navigation Card
- HKI Notification Card
- HKI PostNL Card v1.0.1
- Single resource installation via HACS
- Unified documentation and migration guide
- Backward compatibility with individual card installations

### Notes
- This is the first bundled release combining all five HKI custom cards
- Users can migrate from individual cards without breaking existing configurations
- All card types maintain their original names for compatibility
